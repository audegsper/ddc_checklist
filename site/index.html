Supabase 연결 가이드

  1. `supabase/schema.sql`을 SQL Editor에서 실행합니다.
  2. `site/app-config.js`에 `supabaseUrl`, `supabaseAnonKey`를 입력합니다.
  3. `useSupabase`를 `true`로 바꾸고 페이지를 새로고침합니다.
  4. GitHub Secrets에 `SUPABASE_URL`, `SUPABASE_ANON_KEY`를 저장하면 keepalive 워크플로를 사용할 수 있습니다.
